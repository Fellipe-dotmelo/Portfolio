Principais Habilidades:

DATAVIZ

Desenvolvimento de dashboards interativos e visuais usando ferramentas como Power BI, permitindo uma compreensão clara e rápida dos dados para facilitar a tomada de decisões estratégicas.

ANÁLISE DE DADOS

Implementação de soluções analíticas avançadas, otimizando processos, economizando recursos. Utilizo ferramentas como SQL Server e Power BI para transformar dados brutos em insights valiosos.

PYTHON

Automatização de processos de análise de dados e construção de modelos preditivos com Python, usando bibliotecas como Pandas, NumPy e Matplotlib para manipulação e visualização de dados.

SQL

Desenvolvimento de consultas eficientes e complexas em SQL para extrair, transformar e carregar dados (ETL), otimizando bancos de dados para suportar operações de análise e geração de relatórios.

ETL

Processo envolve coletar dados de várias fontes, transformá-los conforme as necessidades do negócio, e carregá-los em um banco de dados ou sistema de armazenamento para análise.
